# Woolf University. Design Patterns Course. Homework - Final Project

Practice combining multiple design patterns in TypeScript.

## Topic

Build a dynamic resume page using:

* Facade
* Template Method
* Factory Method
* Composite
* Decorator

## Main Logic

The app should demonstrate:

* Loading resume data from JSON
* Validating and mapping data through a template workflow
* Creating resume blocks through a factory
* Rendering nested experience and project blocks
* Highlighting recent projects with a decorator
* Hiding the generation process behind a facade
* Building an extendable HTML resume page

## Run

```bash
npm install
npm run dev
```
